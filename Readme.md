# Amadeus API Backend

This project provides an API for flight booking using the Amadeus API. It supports user registration, login, flight search, and booking. The API documentation is available via Swagger UI.

## Setup

### Prerequisites

- Node.js (v18 or higher)
- MongoDB (or any database you prefer, but MongoDB is used here)

### Install Dependencies

To get started, clone this repository and install the necessary dependencies.

```bash
git clone https://github.com/your-repo/amadeus-api-backend.git
cd amadeus-api-backend
npm install
```

### Environment Variables

Make sure to set up the following environment variables in your `.env` file:

```env
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_jwt_secret
AMADEUS_CLIENT_ID=your_amadeus_client_id
AMADEUS_CLIENT_SECRET=your_amadeus_client_secret
```

### Start the Server

You can start the server using `nodemon` or `node`.

```bash
npm run dev  # For development with nodemon
```

The server will start on `http://localhost:3000`.

### Access Swagger API Documentation

Once the server is running, you can access the Swagger API documentation by visiting:

```
http://localhost:3000/api-docs
```

This will open Swagger UI, where you can view and interact with the API endpoints.

## Endpoints

The API includes the following endpoints:

### `POST /register`
- **Description**: Register a new user.
- **Request body**:
  ```json
  {
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123"
  }
  ```
- **Responses**:
  - 201: User registered successfully
  - 500: Internal server error

### `POST /login`
- **Description**: Log in with an existing user.
- **Request body**:
  ```json
  {
    "email": "john.doe@example.com",
    "password": "password123"
  }
  ```
- **Responses**:
  - 200: Token received
  - 401: Invalid credentials
  - 500: Internal server error

### `POST /search-flights`
- **Description**: Search for available flights.
- **Request body**:
  ```json
  {
    "origin": "NYC",
    "destination": "LON",
    "date": "2025-04-01",
    "adults": 1
  }
  ```
- **Responses**:
  - 200: Flight search results
  - 500: Internal server error

### `POST /book-flight`
- **Description**: Book a flight.
- **Request body**:
  ```json
  {
    "flightDetails": {
      "flightId": "abc123",
      "passengerInfo": { ... }
    }
  }
  ```
- **Responses**:
  - 200: Flight booked successfully
  - 500: Internal server error

### `GET /bookings`
- **Description**: Get user bookings.
- **Responses**:
  - 200: User's booking data
  - 500: Internal server error
